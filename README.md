# Min Stock Management System - DBMS
A Database Management System project that models a simplified stock exchange platform where traders can buy and sell stocks. The project focuses on database design, ER modeling, relational schema, normalization up to BCNF, and SQL database architecture.

## Features

- Trader Management
- Stock Management
- Buy and Sell Orders
- Portfolio Tracking
- Trade History
- Administrator Module
- BCNF Normalization

## Technologies

- MySQL
- SQL
- Database Management System
- ER Modeling

## Database Design

- ER Diagram
- Relational Schema
- Functional Dependencies
- Canonical Cover
- BCNF Normalization
